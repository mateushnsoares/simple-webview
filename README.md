# Simple Webview

## Using

### In development

Run a the dev script:

```bash
npm run dev
```

or using yarn

```bash
yarn dev
```

### In "Production"

Run a script to generate your system executable:

For windows:

```bash
yarn dist:windows
```

For Linux:

```bash
yarn dist:linux
```

For mac:

```bash
yarn dist:mac
```
